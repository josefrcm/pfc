Herramientas usadas para implementar la aplicación.
---------------------------------------------------


Se incluyen paquetes binarios para Windows. En Linux la mayoría de las
distribuciones ya incluyen los paquetes más importantes, pero hay que
compilar a mano Gtk2hs y HaRP. Los pasos a seguir son:

* Gtk2hs (http://www.haskell.org/gtk2hs): 
	* ./configure
	* Editar el Makefile, buscar la opción "-S" (compilar
	  pasando primero a ensamblador), y borrarla. Si no se
	  hace así puede fallar en algunas arquitecturas.
	* make
	* sudo make install

* HaRP (http://www.cs.chalmers.se/~d00nibro/harp):
	* El paquete incluye un Makefile, pero a mí no me funciona.
	* Compilar manualmente cada subpaquete en el siguiente orden:
		* haskell-src-exts
		* harp
		* trhsx
	* Compilar cada paquete con las siguientes instrucciones:
		* runhaskell Setup.hs configure
		* runhaskell Setup.hs build
		* sudo runhaskell Setup.hs install
